Primary Usage

Elementum Teal 3

 

Teal 3 is the Elementum color for primary user action in our palette. It's our color for all text links as well as the color for primary buttons in a resting state.

Elementum Light Gray

 

Light Gray is a secondary - but crucial - Elementum color. It's understated value allows it to been seen when looked for but drop back - like a supporting player - when viewing at the canvas as a whole. It's the background canvas for all our user apps and used for secondary elements that do not require immediate user attention.
